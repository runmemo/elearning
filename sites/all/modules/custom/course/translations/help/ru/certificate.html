После того как студент набирает проходной балл, который определяется как минимальный балл из кредитов курса, ему становится доступен PDF сертификат.

Список доступных вариантов сертификатов показывается студенту в всплывающем окне на странице урока при наведении на баллы студента:

Администратор системы может создавать разные типы сертификатов и назначать их для курсов. Создать новый сертификат можно на странице Добавить сертификат.

Отбражение сертификата включает графический файл с подписью, который можно добавить в поле Подпись:
